# Seed Data

Scripts để chèn dữ liệu mẫu vào databases.

## Sử dụng

Chạy các scripts này sau khi tạo databases để có dữ liệu test:

- `01_users.sql` - Tạo users mẫu
- `02_stations.sql` - Tạo stations mẫu
- `03_chargers.sql` - Tạo chargers mẫu
- ...

## Lưu ý

Chỉ sử dụng cho môi trường development/testing.
Không chạy seed data trong production!
